# 简介

设计药物是与管制物质的结构或功能的类似物，旨在模仿母体药物的药理作用，同时规避法律监管等。但还有许多其他的新型精神活性物质 ，可被视为设计药物的一部分。设计药物亦可称为设计师药物，狡诈家药物等。
